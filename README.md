# treshnanda-taste

A personal **UI/UX taste skill** for Claude Code (and other agents): a soft, premium, pastel, shader-textured, generously-rounded aesthetic paired with **bold, kinetic, Awwwards-tier motion**.

It's the *personal taste* layer that sits on top of two craft skills it **auto-loads** — [`emil-design-eng`](https://animations.dev) (craft details) and `motion-design` (motion theory). One invocation pulls the whole stack.

## Install

```bash
npx skills add Tresnanda/treshnanda-taste
```

Then start a **new** session and invoke it:

```
/treshnanda-taste
```

It will automatically pull in `emil-design-eng`, `motion-design`, and the shared motion vocabulary, then apply the personal taste lens on top.

## What's inside

- **`SKILL.md`** — the taste profile: the quality bar + vocabulary, how I work, the **visual language** (canvas, color, shader surfaces, the image-header card, squircles, type), the **motion doctrine**, copy rules, UX-correctness, and a consolidated **"never" list**.
- **`animation-vocabulary.md`** — a shared glossary of animation patterns (e.g. *shared element transition, morph, rubber-banding, stagger, follow-through, squash & stretch*) so motion requests are unambiguous.

## Use it anywhere

Built to apply to **any app or website**, not one product. Distilled from hundreds of real build iterations plus 33 hand-picked reference designs.
